# Digital Image Processing Final Project
## UNet Brain Tumor Segmentation

The model was trained in the DigitalImageProc_FinalProject.ipynb notebook. This notebook goes through all the steps necessary to download the data, preprocess the data, train a model with the data, and view a test run of the model from the test split. The notebook can also save and load models and optimizers. I intended to upload the model to Github, however, Github apparently has a file size limit, and so I've been limited to keeping it on my Google Drive, link here if you want to download and test it for yourself.
