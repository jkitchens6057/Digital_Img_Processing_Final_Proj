# Digital Image Processing Final Project
## UNet Brain Tumor Segmentation

The model was trained in the DigitalImageProc_FinalProject.ipynb notebook. This notebook goes through all the steps necessary to download the data, preprocess the data, train a model with the data, and view a test run of the model from the test split. The notebook can also save and load models and optimizers. You likely know this, but Github is not an ideal place to train a model, so I've included my trained model and you can then load it in and run the test data through it, if you so desire. I also haven't done any work to make this run in github (things like .env for API keys), mostly because of time constraints. However, github should be able to load the model and test it with the notebook.

## Yolov11 Brain Tumor Segmentation
I've also included the Brain_Tumor_Detection_Yolov11.ipynb notebook for training and testing the Yolov11 version. This is relatively simple in comparison, and I haven't uploaded my model for it, as I don't actually know how to save the model or where it's located. That said, it's easy enough to run and doesn't take nearly as much time or compute as the UNet model does, so if you want to test it, it's not hard.
